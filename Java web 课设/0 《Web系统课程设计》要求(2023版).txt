1、本课设重点考察学生对于HTML、CSS、JavaScript、JSP、Servlet等Web系统开发中用到的技术。为了提高课设质量，特别满足对上述技术已经掌握较好的同学的需求，学院结合企业师资，提供学习视频（链接：https://pan.baidu.com/s/1AhyYS--1ZLMSYfC2EpFa3Q提取码：38t4 ）， 扩展了Vue + axios + Bootstrap技术的学习。该学习资料在机房电脑“软件共享文件夹”中也有备份。建议各指导老师第一天课设时，集中播放序号为0-3的视频。

2、学生根据《Web系统课程设计》指导书（2023）中的要求，3-5人组成项目小组，进行合理分工。项目小组可以选择指导书中题目，也可以自行拟定题目。第一天项目小组将人员和题目告知课设指导老师。。

3.根据分组和选题情况，分别整理个人报告与小组报告。
如遇1人1组的题目，只提交一个个人报告，要求同小组报告；

4.小组报告包含：小组报告封面、任务书、目录、任务书中要求的各类设计文档、心得总结。

5.小组报告中，分节处理各部分内容
 (1) 正文字体为宋体小4，标题为黑体，一级标题2号，2级3号，3级4号;
 (2) 目录部分用罗马数字做页码，目录要求自动生成，页码从正文第1页开始算，从第一章开始；
 (3) 正文中图题格式为"图3-1  xxxx图"，居中放在图下方,表题格式为"表4-1 sss表"，居中放在表上方，表不能分在不同页，其中图中文字要求宋体5或小5号。

6、本课设重点考察学生对于Web系统开发的动手能力，课设报告正文中包含必要的功能需求分析、系统设计、系统实现（系统运行界面截图+相应文字说明+核心代码）、总结与心得部分即可。指导老师验收时，重点验收项目功能完成度。对于使用了Vue + axios + Bootstrap新技术的项目组，建议适当提高成绩。
